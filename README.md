Movie Search Application -- usePopcorn
Overview
The Movie Search Application is a web-based application that allows users to search for movies and view detailed information about them. The application fetches movie data from an external API and displays it in a user-friendly interface. Users can search for movies by title, view movie details, and read user reviews.

Features
Search for movies by title
View detailed information about each movie, including release date, rating, and synopsis
Browse and read user reviews
Responsive design for optimal viewing on different devices
